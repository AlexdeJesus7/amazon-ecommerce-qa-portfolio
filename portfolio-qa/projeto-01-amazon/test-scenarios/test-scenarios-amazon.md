\# Test Scenarios — Amazon Project

\#\# Search Flow

TS-02 — Validate that the system displays results matching the searched term    
TS-03 — Validate access to the product details page from search results    
TS-12 — Validate search behavior with invalid or empty input  

\#\# Product Flow  
TS-04 — Validate product details display    
TS-05 — Validate system behavior when selecting unavailable product variations (color, size, storage, etc.)    
TS-06 — Validate adding a product to the cart    
TS-13 — Validate system behavior when attempting to add unavailable products or variations to the cart  

\#\# Cart Flow  
TS-07 — Validate products visualization in the cart    
TS-08 — Validate updating product quantity in the cart    
TS-09 — Validate subtotal display and informational messages in the cart    
TS-14 — Validate system behavior when reducing product quantity to the minimum allowed  

\#\# Checkout Flow  
TS-10 — Validate checkout access considering user authentication    
TS-11 — Validate total purchase value display after selecting address, payment, and shipping    
TS-15 — Validate mandatory fields in the checkout process    
TS-16 — Validate checkout behavior when changing delivery address to an incompatible location
